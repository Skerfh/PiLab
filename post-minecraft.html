Servidor de Minecraft en Raspberry Pi 5

Introducción

Este proyecto consiste en alojar un servidor de Minecraft en una Raspberry Pi, permitiendo jugar con amigos sin depender de servicios de hosting.

El uso de PaperMC permite mejorar el rendimiento y estabilidad frente al servidor vanilla, algo importante en hardware limitado.

1. Preparación del sistema

Se recomienda instalar Raspberry Pi OS Lite (sin escritorio).


    sudo apt update && sudo apt upgrade -y

2. Acceso remoto (SSH)


    ssh pi@raspberrypi.local

Esto permite administrar la Raspberry Pi sin necesidad de monitor.

3. Instalación de Java


    sudo apt install openjdk-17-jdk -y

4. Descargar servidor

Se recomienda usar PaperMC para mejorar rendimiento y compatibilidad.

5. Ejecutar el servidor


    java -Xms1G -Xmx3G -jar paper.jar nogui

6. Crear servicio (automatización)


    sudo nano /etc/systemd/system/minecraft.service
    

    [Unit]
    Description=Minecraft Server
    After=network.target

    [Service]
    WorkingDirectory=/home/pi/minecraft
    ExecStart=/usr/bin/java -Xms1G -Xmx3G -jar paper.jar nogui
    Restart=always

    [Install]
    WantedBy=multi-user.target

7. Control del servidor


    sudo systemctl start minecraft
    sudo systemctl stop minecraft
    sudo systemctl status minecraft

8. Backups automáticos

En implementaciones reales se utilizan scripts bash y tareas programadas (cron) para generar backups periódicos del mundo.

9. Acceso externo

Para permitir conexiones externas se puede usar port forwarding o túneles como Pinggy o ngrok.

Conclusión

Este proyecto permite aprender Linux, redes y automatización, además de crear una infraestructura real sobre Raspberry Pi.
