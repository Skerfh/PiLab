Bloqueador de anuncios en red con Pi-hole

Introducción

Pi-hole permite bloquear anuncios a nivel de red actuando como servidor DNS. Esto hace que todos los dispositivos se beneficien automáticamente.

¿Cómo funciona?

Cuando un dispositivo intenta acceder a un dominio de anuncios, Pi-hole bloquea la solicitud DNS.

1. Instalación del sistema

Instalar Raspberry Pi OS Lite.

2. Acceso SSH


        ssh pi@raspberrypi.local

3. Actualización


        sudo apt update && sudo apt upgrade -y

4. Instalación de Pi-hole


        curl -sSL https://install.pi-hole.net | bash

5. Configuración

  • Seleccionar red (Ethernet)
  • Elegir DNS upstream (Cloudflare recomendado)
  • Activar panel web

6. Acceso al panel


        http://pi.hole/admin

7. Configuración del router

Configurar la IP de la Raspberry Pi como servidor DNS principal para aplicar el bloqueo en toda la red.

Resultados

  • Menos anuncios
  • Menos tráfico de red
  • Mayor privacidad

Conclusión

Pi-hole es uno de los proyectos más útiles y fáciles de implementar en Raspberry Pi, con impacto inmediato en toda la red.
